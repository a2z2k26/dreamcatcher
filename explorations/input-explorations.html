Accessibility
All animations respect prefers-reduced-motion. When reduced motion is active, transitions collapse to instant state changes with a 1ms duration (perceived as immediate). Continuous animations (cursor blink, streaming pulses, ember particles) stop entirely. Focus indicators remain visible and unaffected.
Context menu items are keyboard-navigable. Tooltips use appropriate ARIA roles (role="tooltip"). Destructive actions use color + icon + separation for triple redundancy -- never color alone. All flash rates stay below 3 per second.

# Green-Land Agent Briefs

Use these briefs when spawning specialist agents for a Green-Land loop. Give every agent:

- the latest `artifacts/greenland/<timestamp>/index.html` or screenshots
- `docs/design/greenland/rubric.md`
- relevant reference files under `Dreamcatcher Design/` and `Red Stripe Design System/`
- the exact slice under review

Agents should return concrete findings, not taste essays. Implementation agents must use disjoint write scopes.

Recommended swarm for one loop:

1. `design-chief.md`
2. `design-system-architect.md`
3. `visual-designer.md`
4. `interaction-designer.md`
5. `accessibility-specialist.md`
6. `frontend-implementer.md`
7. `qa-visual-reviewer.md`

# Docs

Supporting research, specs, and reviews for Dreamcatcher. The product spec
and strategy documents live at the repo root (`../PRODUCT.md`, `../STRATEGY.md`).

## Layout

### `design/`
Visual language, motion, and design system.

- `DESIGN-SPEC.md`, `DESIGN-SYSTEM.md`, `DESIGN-ROADMAP.md`, `DESIGN-AUDIT-2.md`
- `VISUAL-MANIFESTO.md`, `VISUAL-CRITIQUE.md`, `NODE-VISUAL-SYSTEM.md`
- `TOKEN-ARCHITECTURE.md`, `MOTION-SPEC.md`
- `DELIGHT-SPEC.md`, `DELIGHT-RESEARCH.md`
- `DARK-UI-RESEARCH.md`, `DARK-UI-PATTERNS.md`, `DARK-THEME-REFERENCES.md`

### `engineering/`
Non-feature engineering playbooks.

- `ACCESSIBILITY-GUIDELINES.md`
- `PERFORMANCE-PLAYBOOK.md`
- `POLISH-CHECKLIST.md`
- `SPRINT-PLAN.md`

### `audits/`
Point-in-time reviews of the product and codebase.

- `AUDIT-ROUND-3.md`
- `UX-AUDIT.md`
- `HONEST-ASSESSMENT.md`
- `QUALITY-SCORECARD.md`
- `INTEGRATION-REPORT.md`

### `principles/`
Foundational principles guiding the work.

- `GUIDING-PRINCIPLES.md`
